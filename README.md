Smart screen locking with Facial Verification.


## Requirements

- Python3
- Tensorflow (Python3)
- Gmail account
- BASH (as a default shell)

## Installation

```
$ git clone https://github.com/Quimzy/nanolock.git

$ pip3 install requirements.txt

$ python recognizer.py
```
